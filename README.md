# 🤖 AI Face Detection System

An Artificial Intelligence Face Detection application built using OpenCV and Streamlit.

## 🚀 Features

- Detect faces from uploaded images
- Real-time webcam face detection
- Face counting
- AI-style user interface
- Download processed image
- Responsive and modern design

## 🛠 Technologies Used

- Python
- OpenCV
- Streamlit
- NumPy
- Pillow
- Streamlit WebRTC



## ▶ Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
